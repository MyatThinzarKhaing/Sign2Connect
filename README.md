# Myanmar Sign Language Gesture-to-Text Recognition

## Description
Real-time gesture-to-text system for Myanmar Sign Language (43 dynamic gestures).  
Uses CNN (ResNet50) + Transformer for feature extraction and sequence modeling.

## Features
- gesture recognition with Flask
- Data preprocessing, augmentation, and motion feature extraction
- Kaggle notebooks for preprocessing and training

## Project Structure
- `app.py`: Flask web app
- `model/`: Saved trained model
- `notebooks/`: Kaggle notebooks
- `utils/`: Helper functions
- `requirements.txt`: Python dependencies
